### Редактирование текста в PDF

1. Откройте PDF-файл (Файл → Открыть)
2. Нажмите **"Редактировать PDF"** на панели инструментов
3. Кликните по тексту, который хотите изменить
4. Напечатайте новый текст
5. Измените шрифт, размер или цвет (если нужно)
6. Нажмите **"Сохранить"** (Ctrl+S)

### Редактирование изображений

1. В режиме редактирования кликните по изображению
2. Нажмите правой кнопкой → **"Заменить изображение"**
3. Выберите новое изображение на компьютере
4. Или используйте **"Переместить"**, чтобы изменить положение
5. Измените размер, перетаскивая углы

### Конвертация PDF в Word

1. Нажмите **"Конвертировать PDF"**
2. Выберите **"PDF в Word"**
3. Нажмите **"Выбрать файл"** и укажите PDF
4. Нажмите **"Конвертировать"**
5. Дождитесь завершения
6. Откройте DOCX-файл в Microsoft Word

### Объединение нескольких PDF

1. Нажмите **"Инструменты"** → **"Объединить файлы"**
2. Нажмите **"Добавить файлы"**
3. Выберите все PDF, которые нужно объединить
4. Перетащите файлы для изменения порядка
5. Нажмите **"Объединить"**
6. Сохраните результат

### Добавление цифровой подписи

1. Нажмите **"Заполнить и подписать"**
2. Нажмите **"Подпись"** → **"Создать подпись"**
3. Нарисуйте, введите или загрузите изображение подписи
4. Нажмите **"Создать"**
5. Перетащите подпись в нужное место документа
6. Сохраните PDF

### Защита PDF паролем

1. Нажмите **"Инструменты"** → **"Защита"**
2. Выберите **"Зашифровать"** → **"Зашифровать паролем"**
3. Введите пароль дважды
4. Выберите разрешения (печать, копирование и т.д.)
5. Нажмите **"ОК"**
6. Сохраните PDF — теперь он защищён
